🎄 Emerald Christmas Magic | 祖母绿沉浸式圣诞树
一个极简奢华风格的 Web 3D 圣诞互动体验项目。结合了 Three.js 粒子系统、MediaPipe 手势识别以及 Canvas 烟花引擎，打造出电影级的视听盛宴。

✨ 特别优化：本项目已针对 iPad 和 Android 平板 进行深度适配，支持前置摄像头手势控制与触摸交互。

🌟 核心功能 (Features)
🎨 视觉体验
祖母绿 & 流金主题：采用深祖母绿背景搭配高光金色 UI，营造极简奢华的节日氛围。

3D 粒子圣诞树：由 3000+ 个动态粒子构成的三角锥形树体，包含树干、深绿树叶、呼吸灯光（红/黄/蓝/紫/白）及顶部的伯利恒之星。

电影级特效：集成了 Unreal Bloom（辉光）后处理效果，使灯光和烟花具有朦胧的唯美感。

动态烟花：基于 Canvas 的高性能烟花系统，包含柳树状和彗星状烟花，随场景交互自动触发。

🖐️ 交互与手势 (Gesture Control)
基于 MediaPipe Hand Landmarker 的实时手势识别系统（带防抖动锁定机制）：

🖐️ 张开手掌 (Open Palm)：将圣诞树“炸开”成旋转的粒子云，照片在其中盘旋展示。

👌 双指捏合 (Pinch)：在粒子云模式下，随机抓取一张照片并放大到屏幕中央（类似盲盒抽取）。

✊ 握拳/收手 (Fist)：将所有粒子重新凝聚成圣诞树形状。

📱 设备适配
平板优化：自动检测移动设备，调整粒子数量（3200 -> 1600）以保证 60fps 流畅运行。

沉浸模式：点击右上角“眼睛”图标或按 F 键，隐藏所有 UI 和摄像头画面，只保留纯净的 3D 场景。

移动端修复：解决了 iOS/Android 浏览器地址栏遮挡问题（使用 dvh 单位），强制调用前置摄像头。

⚙️ 自定义配置
照片墙：用户可上传多张照片，生成悬浮在 3D 空间中的金色相框。

背景音乐：支持上传自定义音频文件。

节日祝福：可自定义主标题文字（如 "Happy New Year"）。

🎮 操作指南 (User Guide)
1. 初始化：打开网页，点击 "Enable Camera Access"（或 Start Magic）。

2. 设置：

输入你的祝福语（默认 Merry Christmas）。

点击 "Select Images" 上传几张照片。

（可选）上传背景音乐。

点击 Ignite Magic 进入主场景。

3. 手势控制：

将手举起，确保摄像头能看到完整的手掌（左下角有预览框）。

张开手：树木散开。

捏合食指与大拇指：抓取照片。

放下手：树木复原。

4. 沉浸模式：点击右上角的 👁️ 图标，享受纯净画面。
